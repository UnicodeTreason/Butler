# Butler

A tool to perform some common tasks based on Key Shortcuts.

## Usage

- Ctrl + q (Quit App)
- Ctrl + Alt + v (Click mouse and type out clipboard, useful for pasting into RDP Sessions)
- Ctrl + UpArrow (Scroll Mouse Wheel Up a Lot)
